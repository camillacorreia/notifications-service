## 🚀 Tecnologias

This project was developed with the following technologies:

- NestJS
- TypeScript
- Jest
- Prisma
- Kafka

## ℹ️ How To Use

```bash

# Install dependencies
$ yarn

# Run Prisma Generate
$ yarn prisma generate

# Start server
$ yarn dev

# running on port 3000
```
